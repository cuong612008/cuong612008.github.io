Guidelines

Do's & Don'ts

Principles for consistent application of the Nguyễn Đỗ Cao Cường design language across all digital surfaces.

Do — Embrace
  • Use Deep Charcoal (#252121) as dominant primary text and UI color
  • Apply Burnt Orange (#F26822) sparingly for critical CTAs and accents
  • Maintain high contrast between text and background for accessibility
  • Employ generous whitespace to convey confidence and clarity
  • Use Montserrat for UI and Source Serif Pro for navigation/links
  • Default buttons to 25px pill-shaped with 40px height
  • Stack sections vertically with alternating light backgrounds
Don't — Avoid
  • Mix serif and sans-serif within the same line of text
  • Use orange as a background color for extended text areas
  • Reduce button padding below 12px 30px or border radius below 25px
  • Stack more than 2-3 levels of heading on one page
  • Apply shadows that create a plasticky, heavy appearance
  • Forget to test form inputs and buttons for mobile touch targets
  • Deviate from the 4px spacing grid for arbitrary values
  • Use more than 2 accent colors on a single page simultaneously
